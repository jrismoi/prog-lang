key signature
key: C_major
key: G_major
key: A_minor

Must appear at the top of the file. Sets the scale for all note degrees.

note syntax
octave.degree[+/-].duration[suffix]

1.3.4    → octave 1, 3rd degree, quarter
0.5.8    → octave 0, 5th degree, eighth
-1.1.1   → octave −1, root, whole

Octaves are keyed to the tonic. In D_minor, octave 1 spans D4 up through C5; octave 0 spans D3 through C4.

accidentals
1.3+.4   → sharp  (E# in C major)
1.3-.4   → flat   (Eb in C major)

Append + or − directly after the scale degree number.

durations & suffixes
1        whole note      (4 beats)
2        half note       (2 beats)
4        quarter note    (1 beat)
8        eighth note     (½ beat)
16       sixteenth note  (¼ beat)

4.       dotted quarter  (1.5 beats)
4..      double-dotted   (1.75 beats)
4t       quarter triplet (⅔ beat)
8t       eighth triplet  (⅓ beat)

Dot suffixes multiply duration by 1.5 or 1.75. The t suffix compresses three notes into the space of two.

rests
R.4      quarter rest
R.8      eighth rest
R.2      half rest
R.4.     dotted quarter rest

Use R (capital) in place of octave.degree to insert a silent rest of any duration.

repeats
Rep2 {
  1.1.4
  1.3.4
  1.5.4
}

Rep3 {
  1.5.8
  1.3.8
}

Rep followed by a count, then curly braces. The contents play that many times in sequence.

comments
// this is a comment
1.1.4  // inline comment

Use // to annotate your score. Everything after // on a line is ignored and rendered softly in the editor.

time signature

Declare after key. Adds measure barlines to the output and shows a live measure helper while you type in the source pane.

key-based octaves
// Octaves are anchored to the key root.
// In D_minor, octave 1 runs from D4
// up to the next D:
1.1.4   // D4
1.7.4   // C5
0.7.4   // C4

Octave numbers always span from the key's tonic up to the note below the next tonic, rather than resetting at C.
